# PWA 아이콘

PWA 설치 시 사용되는 아이콘입니다. 다음 두 파일이 필요합니다:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512)

## 임시 생성 방법

배포 전에 실제 로고 PNG를 만들어 이 폴더에 넣어주세요. 권장 디자인:
- 배경: `#0f1b2d` (네이비)
- 로고: `SCAN&GO` 흰색 + `&GO` 부분만 `#00c9a7`
- 충분한 여백 (maskable 대응 — 가장자리 10% 안전영역)

## 즉시 사용 가능한 placeholder

피그마/일러스트레이터가 없으면 https://realfavicongenerator.net 또는 https://maskable.app/editor 에서 빠르게 생성 가능.

생성 후 두 PNG 파일을 이 폴더에 직접 넣으면 됩니다.
